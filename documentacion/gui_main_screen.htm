User Manual

Step-By-Step Guide to understand the program.

1. Abrir el programa y presionar en "insert a coin".

2. Ingrese la cantidad de pilas (de colores que posee su archivo) y posteriormente la capacidad que desee darle a las pilas, recuerde que si no concuerda la cantidad de pilas con su archivo de texto, no podrá seguir con el programa. Despúes de su selección, presione el botón de comprobar, posteriormente el de "ingresar" y se le abrirá una nueva pestaña donde podrá ingresar su archivo .txt con el formato de los colores, si el archivo es válido, aparecerá el botón de siguiente, el cual debe presionar.

3. Se le abrirá la siguiente pestaña donde podrá elegir si desea jugar con pausas (donde el usuario mueve los bloques de colores) o sin pausas (donde el programa realiza el ordenamiento). Si selecciona con pausas le preguntará la dificultad y posteriormente se desbloqueará el botón "GO" el cual debe presionar, si seleccionó sin pausas ir al paso 12.

4. Se le abrirá una nueva pestaña con el archivo ya cargado y los bloques desordenados, dependiendo la dificultad que haya elegido así tendrá el limite de movimientos, donde en fácil tendrá movimientos limitados, medio 50 movimientos y díficil solamente 25, si sobrepasa dichas cantidades el juego se reiniciará. Para poder jugar primero debe presionar el método de ordenamiento que gusta entre modo "cola" o modo "pila".

5. Después de elegir su modo, puede elegir la lista de donde desee obtener el dato.

6. Después de elegir su modo, puede elegir la lista de donde desee sobreponer el dato.

7. Si se puede mover la pieza de la manera que seleccionó, esta se moverá, de lo contrarió lo hará seleccionar otra lista de destino.

8. Al momento de haber terminado de ordenar su lista, tiene la opción de comprobar la victoria, exportar los pasos o bien en cualquier momento de guardar su partida con el botón de save file.

9. Si selecciona el botón de comprobar victoria a este punto, le confirmará su victoria, de lo contrario, le dirá que continue con el juego.

10. Si desea guardar los pasos puede presionar el boton de exportar pasos y le abrirá una nueva pestaña donde podrá elegir en donde guardar el archivo de texto, posteriormente le confirmará si se guardó su archivo o no en caso de algún error.

11. Si desea guardar el estado actual de la partida, puede presionar el botón save file, donde posteriormente le abrirá una nueva pestaña donde podrá elegir el directorio de guardado del archivo de texto, posteriormente le confiramrá si se guardo su archivo o no en caso de algún error.

12. Si seleccionó el modo sin pausas, entonces le preguntará si desea hacer el ordenamiento por cola o pila y posteriormente se le activará el botón "GO" el cual debe de presionar.

13. Se le abrirá una nueva pestaña con el tablero desordenado, donde puede visualizar lo siguiente.

14. Presione el botón ordenar para ordenar el tablero automáticamente.

15. Presione el botón de exportar pasos si desea ver los pasos, posteriormente le pedirá una ruta donde guardar el archivo de texto.
